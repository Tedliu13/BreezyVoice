NCDR 2.1.4專案 Text-To-Speech服務

上傳Vocal音檔與對應文字作為生成語音的聲音，批次 CSV僅需上傳一行標題為content_to_synthesize，內容為要轉換語音的文字，網頁會呼叫同一台服務上的 API，將所有文字批次轉成 WAV，最後自動打包成 ZIP 供你下載。

批次輸入

支援欄位：`text`、`content_to_synthesize`、`content`、`input`。可選檔名欄位：`filename`、`output_audio_filename`、`output_filename`、`id`、`name`。

CSV 範例

最簡單的格式只要有一個文字欄位即可。若你要自訂輸出檔名，可以多放一個 `filename` 欄位。
filename,text intro,今天天氣真好 alert,冷氣團南下 北部轉涼白天氣溫降八度 demo,歡迎使用 BreezyVoice 的批次文字轉語音功能
API 端點為 /v1/batch/speech。同一頁面下載完成後，ZIP 內會包含所有 WAV 與一份 manifest.csv。
